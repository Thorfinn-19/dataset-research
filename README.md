Ez a tárhely python scripteket  tartalmaz szintetikus kártya adatkészletek létrehozásához objektumdetektálási kísérletekhez. 
Ezenkívül tartalmazza a létrehozáshoz szükséges kártya png képeket (assets/cards), 
mintákat a hátterekről (assets/sample texture) amelyekre a kártyák elhelyzésre kerülnek, 
mintákat az elkészült adatkészletek képeiből és a hozzájuk tartozó címkékből,
és a majd a tanításhoz szükséges .yaml fájlokat.
